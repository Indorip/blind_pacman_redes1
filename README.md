Repositório para o trabalho da matéria de Redes 1 do curso de BCC na UFPR 2026, semestre 1.

Trabalho consiste na implementaçáo do jogo pacman em um sistema cliente-servidor.

Protocolo para pacotes é um kermit-like. Foi utilizada uma raw-socket promiscua para envio dos pacotes entre computadores.
O programa funciona entre computadores ou em loopback no mesmo computador e realiza envio dos arquivos 1 .. 6 ao jogador comer os pellets.

Forma de uso na linha de comando e mais detalhes da implementação no relatório do projeto.
